---
title: Opener
description: Open URLs and files or reveal a path in the system file manager.
---

```bash
carbon add plugin Opener
```

```ts
import { opener } from '@dotcarbon/plugin-opener'

await opener.openUrl('https://dotcarbon.dev')
await opener.openPath(reportPath)
await opener.revealPath(reportPath)
```

Use Opener for user-facing actions that delegate to the operating system. It does not provide arbitrary
process execution; use the more tightly configured Shell plugin when a child process is required.
