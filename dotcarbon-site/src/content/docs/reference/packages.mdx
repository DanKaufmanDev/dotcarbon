## NuGet

| Package | Purpose |
| --- | --- |
| `DotCarbon.Core` | Runtime, bridge, source generator integration, config, security |
| `DotCarbon.Cli` | Global `carbon` .NET tool |
| `DotCarbon.Host.Desktop` | Photino desktop host, tray, and menus |
| `DotCarbon.Host.Android` | Android WebView host |
| `DotCarbon.Host.iOS` | WKWebView iOS host |
| `DotCarbon.Plugins.*` | First-party backend plugins |
